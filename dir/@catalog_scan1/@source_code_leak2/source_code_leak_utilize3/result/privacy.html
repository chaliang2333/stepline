屋瓴隐私服务

上海屋瓴网络科技有限公司所提供的网站（域名为wulingd.com、wuling.me、wujar.com）及移动APP（IOS及安卓终端的金牌经纪人、金牌顾问、屋瓴通、屋瓴数据全景展播），以上均简称为我们。我们理解及关心您的信息、数据如何被获取及使用。本协议将阐述我们通过上述产品，对用户隐私的获取及保护策略。如您使用屋瓴产品，注册并同意我们的隐私及服务内容，则视为您同意、接受我们关于您的隐私相关协议。

我们通过向您提供包含网站、APP、短信、邮件等服务，获取到您的相关客户信息，仅用于提升用户体验，打造个性化服务。我们所采集的用户信息如下：

您向我们提供的信息：我们收到并存储您在我们网站上、APP上及其它产品上，以任何形式提供给我们的任何信息。您可以选择不向我们提供全部或某些个人信息，但同时您也将无法享受到屋瓴更多的产品及服务。我们所收集您的信息，仅用来更好的提升我们的产品及体验。

自动获取的信息：无论您何时与我们进行沟通，我们接受并储存特定的信息。例如我们会通过COOKIES，来获取您在浏览页面及广告时的某些信息。

移动设备：当您使用屋瓴网提供的各项APP应用时，我们将收集您的位置信息及手机硬件信息，其中包含识别您的移动设备的特定号码。我们使用这些信息为您提供基于位置的定位相关服务，例如广告，搜索结果以及其它个性化内容展示。大多数的移动设备都允许您关闭定位服务。

内容用以保证某些流程正常进行下去，我们确保在需要获取您短信及邮件内容时，会及时告知于您。我们在对您触发短信及邮件联系前，会谨慎存储及使用您的联系信息，避免您收到垃圾内容。

其他来源：我们仍将可能通过其他方式获取您的信息，并加入到屋瓴大数据体系中。

用户信息共享

用户信息的收集及应用，是屋瓴大数据体系中非常重要的一部分。为此，保护用户隐私信息，不丢失，不泄露，不转卖，是我们用户信息共享使用的原则。但由于用户部分信息是通过屋瓴其它商业合作伙伴所获取的，我们仅以下述方式与适用本隐私声明或至少提供与本隐私声明同样保护性惯例的本网站关联公司分享客户信息。

不受我们控制的商业伙伴：对于那些在屋瓴投放广告或有其它非系统性合作的商业伙伴中，由于广告投放涉及到cookies信息、报名、付费等个人信息，我们将与其共同分享您的个人信息

第三方服务提供者：我们会使用其它第三方的公开服务，如第三方短信电邮平台、第三方支付（线上及线下支付）、第三方呼叫中心等。此类第三方公司会接触到您的个人信息，但商用第三方平台已与我们签订相关的协议，保证客户信息不作其他商业目的使用。

业务转让：为持续发展我们的业务，我们可能会收购部分地产运营、代理、地产中介等企业。在这些交易中，客户信息通常是所转让企业资产中的一部分，但这些信息（当然除非客户作出其他同意）仍然受制于转让前已有的任何隐私声明所作出的承诺。虽然可能性不大，但在极端的情况下，如果本网站或其关联公司或他们各自的全部资产被收购时，客户信息应是被转让资产的一部分。

对本网站和其他人的保护：为了遵守法律、执行或适用我们的使用条件和其他协议，或者为了保护本网站、我们的客户或其他人的权利及其财产或安全，我们将会提供帐户或其他个人信息。这包括为防止欺诈等违法活动和减少信用风险而与其他公司和组织交换信息。不过很明显，这并不包括违反本隐私声明中所作的承诺而为获利目的出售、出租、共享或以其它方式披露个人可识别的信息。

征得您的同意：除以上规定之外，当有关您的信息有可能披露给第三方时，您将会得到通知，并且您将有机会选择不与第三方分享此信息。

关于您的信息安全

我们在对您的账号密码信息校验时，会通过加密方式进行，努力保证您的账号安全。

关于您的银行账号等信息，屋瓴网通过有互联网支付资质的第三方支付公司完成您的卡号记录及审核。

在界面展示上，我们仅展示您的后四位卡号信息。防止他人未经授权使用您的密码或进入您的计算机是非常重要的。因此，您应确保在与他人共用一台计算机或移动设备时，使用完毕后即时退出登录。

其它关于隐私保护内容

网站上的第三方广告和链接：我们的网站及APP上可能会出现第三方广告及其它非屋瓴网的链接地址，有时使用技术手段将我们网站上显示的广告直接发到您的浏览器上。这时它们自动获取您的IP地址。它们也可能使用cookies, JavaScript, web beacons等技术来衡量它们广告的效果以及个性化广告内容。我们无从控制它们使用的这些cookies或其它技术，而且本隐私政策不适用于这些网站或广告商所从事的行为。请与它们直接联系以获得关于它们隐私政策的详细情况。

本网站允许您进入与您的账户以及和本网站进行互动有关的信息，这种访问的目的仅限于阅读和在特定情形下更新账户信息。 相关信息将随网站的变化而更新。您在我们网站上发布信息时，如果我们发现您发布法律、法规禁止发布或者传输的信息，我们有权立即停止传输该信息，采取消除等处置措施，保存有关记录，并向有关主管部门报告。如果有其他客户发现您发布的信息泄露个人身份、散布个人隐私等侵害其合法权益，或您如在我们许可的情况下发布商业性电子信息，而其他客户表示遭受侵扰，这些客户有权要求我们删除您发布的信息，或者采取其他必要措施予以制止。我们将会依法采取这些措施。
