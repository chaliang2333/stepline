关于屋瓴海外

屋瓴海外置业是上海屋瓴（www.wuling.me）旗下，高端客户全球不动产配置中心，帮助中国大陆中产者进行海外资产配置，我们主要涉及欧美高端移民、大洋洲尊享移民，以及东南亚置业投资。

屋瓴马来置业致力于成为中产阶级首次海外置业领域的领跑者，为有首次海外资产配置需求的客户提供最专业的指导和最贴心的服务。

我们与马来排名前五地产公司——房天地产（www.fangtian.com.my）建立深度合作关系，成为其全球战略大中华区的唯一合作伙伴。通过强强联手，为广大马来置业意向客户提供最周全配置方案，最优质房源，总有一项符合客户心愿。
