屋瓴常见问题答疑

聆听·评估·反馈

屋瓴网和某多多的区别是什么？

屋瓴网和某多多的共同点均在于通过整合地产营销的外场资源，提升客户量，更好的服务于地产营销。

不同点在于，屋瓴网通过提供屋瓴云商系统，帮助全国各地的地产运营商、代理商完成场内外资源的整合。屋瓴在该过程中，不仅整合经纪人渠道、还提供外场媒体、渠道公司、其它网站、老客户、意见领袖等在内的其它获客渠道的资源整合。将获客--成交--结算流程化，满足所有客户的使用需求。

此外，屋瓴作为房产电商系统解决方案的提供商，支持各地客户自立品牌，自建门户。我们不设城市分公司，不涉及任何城市地产营销业务，没有直营或加盟的经营方式。

屋瓴网及屋瓴云商系统是什么？

屋瓴网是屋瓴云商系统中的一部分。

屋瓴云商系统是指由云服务、外场业务系统、内场业务系统、POS对单系统、清算系统、大数据应用系统、广告投放系统等在内的所组成的房产电商营销系统。屋瓴网站包含屋瓴官网（wuling.me）、屋瓴租户管理后台（wujar.com）、屋瓴管理后台（wulingd.com）等站点。

我要使用你的系统，但入门会不会很难？

如果您已经拥有合作中的外场资源，如经纪公司，渠道公司，小蜜蜂派单员，网站等资源。仅需2周，就可完成系统角色的安排、试运行及上线。如果您尚未有合作中的外场资源，我们会提供对应的促成方案和建议，助力您整合资源，预计可在30-45天中完成所有资源配备，系统上线运行。

以上的两种情况，我们会配备相应的工作人员，驻场协助及培训。帮助您顺利完成系统落地和熟练应用。

如果您有更多问题，欢迎致电我们021-31660706，或扫码咨询屋瓴业务经理，了解更多。

我的客户信息会不会泄露或被使用？

屋瓴只提供系统，不涉及任何城市的地产营销或客户经营业务。

屋瓴云商系统所使用的所有服务器均架设在阿里云上，通过阿里所提供的专业服务，我们的服务器安全性堪比淘宝。在系统安全性上，我们对所有敏感信息进行脱敏和加密处理，操作日志定期巡查和异常报警；内部风控上我们对权限申请，敏感信息触碰、提取及采集均有对应的流程，三重防护，保护客户的信息安全。

什么是POS配单服务，我要如何申请使用？

屋瓴POS配单系统，是指所有通过屋瓴云商系统记录在案的客户，通过屋瓴提供的POS机刷卡或第三方在线支付在认购时，屋瓴系统会自动识别并将客户状态标记为成交客户，并根据此来完成清算工作。

目前我们所提供的POS机及第三方在线无卡支付，均使用的是上海汇付天下相关产品和软件。屋瓴POS配单服务是屋瓴云商系统中的一部分，成为屋瓴的合作伙伴，即可享受该项功能及服务。
