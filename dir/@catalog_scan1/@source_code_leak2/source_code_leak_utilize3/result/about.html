公司简介

上海屋瓴网络科技有限公司，努力践行 “华屋其势，建瓴云商”的信念。来自全国各地房产行业、互联网行业及金融行业的优秀人士组成了屋瓴大家庭，公司通过向房地产运营商提供系统化软件服务，整合地产营销内外场资源，建立包含房地产营销公司、开发商、外场渠道公司、经纪人、购房客户、媒体等在内的生态体系，集合数据资源，最终实现以大数据技术服务全地产行业。

技术团队

目前技术中心已经组建DEV、UED、PM、SDC等多个研发小组，涵盖产品研发、ui设计、前端、后台、移动端、测试，到最终产品上线等一系列开发流程，团队成员最少都拥有5年以上的IT行业经验，核心技术成员多来自于平安、腾讯、百度等国内品牌互联网公司。

屋瓴时光机

至2015年11月，仅用5个月时间，我们完成百万行代码的编写，旗下10大产品全部上线，业务覆盖城市50余个。12月31日，屋瓴完成首轮融资，由国内知名投资机构零一创投和经纬中国完成投资，投资额千万级。

运营团队

为了加强对运营商的支持与服务力度，公司还组建了专业的运营支持团队，其中包括产品培训、业务培训、客户服务、数据分析等多个部门，力求给予运营商最强有力的运营保障。“产品交付只是合作的开始”，这是屋瓴SaaS服务相较于其他传统技术开发公司最大区别。
