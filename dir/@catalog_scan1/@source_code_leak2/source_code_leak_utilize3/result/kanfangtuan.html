行程路线

Day 1 ：出发 - 吉隆坡
我们挑选最优质的酒店，让您尽情享受整个吉隆坡之旅，您将入住铂尔曼国际酒店(Pullman)或同级酒店。
*铂尔曼国际酒店：酒店身处繁华市中心，步行前往双子塔仅10分钟，毗邻巴比伦购物广场，您可以在逛街之余，品尝特色娘惹菜。

Day 2 ：独立广场 - 吉隆坡城市馆 - 占美清真寺 - 双子塔 - 黑风洞 - 自由行
我们将带您游览吉隆坡一系列最受欢迎的景点，结束精彩行程，意犹未尽的您仍可以选择继续自由游览。

Day 3 ：参观项目 - 自由行
充分的休息后，您将实地考察马来西亚极具吸引力的热门项目：雅乐居项目(Agile)、星之宿项目(Star)。
雅居乐身处吉隆坡高档成熟社区——满家乐，周边环绕英式及美式的顶级国际学校。商住一体的星之宿项目，生活便利，精英群聚。
您还能观赏马来西亚最高元首，联邦苏丹王的壮丽官邸——国家皇宫，并自由安排空余行程。

Day 4 ：参观项目 - 自由行
您将参观双子塔旁的市中心高端项目——国齐轩(Isola)。随后前往吉隆坡艺术殿堂——中央艺术坊，这里聚集着充满东南亚民俗风情的小工艺品。离开中央艺术坊，便可以去茨厂街体验“吉隆坡的唐人街”，闲暇中尽情享受自由行的乐趣。

Day 5 ：自由行 - 回城
最后一天，我们的同事会在您飞机起飞前两小时将您送到机场。在这充裕的时间内，您可以任性买买买、吃吃吃。如果您通过此行爱上了大马，可以让我们帮您在此安家。
